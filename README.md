How to Run the Bot

Follow these simple steps to set up and run the Automatic Email Sender Bot:

1. **Configure Credentials**: Open `emailsender.py` and input your sender email and your 16-character Google App Password inside the configuration variables.
2. **First Run**: Open your Command Prompt (CMD) or Terminal, navigate to the project folder, and run the following command:
   py emailsender.py
   Automatic File Creation: The bot will notice that the configuration files are missing and will automatically generate two text files in your directory:

subject.txt

message.txt

Set Your Subject: Open subject.txt, type the subject of your email, save it, and close the file.

Write Your Message: Open message.txt and type your complete email body. Feel free to use multiple lines, paragraphs, and spaces. Once done, save and close it.

Start Sending: Run the bot again using the same command:
py emailsender.py
Quick File Attachment (Drag & Drop): When the bot asks if you want to attach a file, type yes or y. Then, simply drag your file from your desktop and drop it directly into the terminal window. The bot will automatically capture the exact file path and attach your document (e.g., CV, PDF, or portfolio) instantly!
